PORTFOLIO

Jumpstart

Jumpstart is a full stack React application designed for software developers of all experience levels to post about questions and accomplishments, ask for help, and comment on posts of other developers. Versions 2 and 3 will include markdown support for posts and file storage, for users to remotely store their repositories. Check it out here!

CineBuster

CineBuster is a full stack e-commerce built with React and with Stripe integration for payment authorization and acceptance. Users are able to add and remove to and from their shopping cart, purchase from their cart, and review or request a refund for movies once purchased. Check it out here!

HouseMates

HouseMates is a full stack application built with JavaScript, HTML, and CSS on the front end, and with Express API on the backend where users can post to find roommates in their given location. Check it out here!
